# 2026 PGA of BC Sponsor-Media Invitational

Single-page website for the 2026 PGA of BC Sponsor-Media Invitational at University Golf Club.

## 📁 Folder Structure

```
/
├── index.html        ← Main site file
└── images/
    ├── README.md     ← Full list of required images
    ├── img_001.jpg   ← (add your images here)
    ├── img_002.jpg
    └── ...
```

## 🖼️ Adding Images

All images go inside the `images/` folder. See [`images/README.md`](images/README.md) for the full list of required filenames and what each one is used for.

## 🚀 Running Locally

Just open `index.html` in a browser — no build step required.
